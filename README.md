# Rocket Boost

Rocket Boost — аркадная игра на Unity, в которой игрок управляет ракетой и должен безопасно приземлиться на платформу.

##  Геймплей
Игрок управляет тягой и вращением ракеты, избегает препятствий и проходит уровни, приземляясь на целевые платформы.

##  Реализовано
- Управление через Rigidbody
- Физическая модель движения
- Обработка столкновений
- Система победы и поражения
- Переключение сцен
- Звуковые эффекты
- Частицы

##  Технологии
- Unity
- C#
- Rigidbody Physics
- Scene Management
